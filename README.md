# The Tomb of Ocura
You awaken in a dark cell. There is a bed in your cell, and a silver key on the ground. Outside your jail cell, the room has a large pile of hay, and an exit to the north.

What will you do?

---

The Tomb of Ocura is an RPG text-based adventure, taking inspiration from games such as Dark Souls, Pokemon, and Zork.

Using simple commands, explore a deep dungeon, fight monsters, level up and become more powerful, while finding our the truth behind the ancient tomb deep in the earth of Ocura. 

## About this Game
This game is develped in C#, using a base of a design project for school, but converted into C# from Python.
